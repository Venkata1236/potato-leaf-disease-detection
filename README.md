# Potato Leaf Disease Detection

This project is an AI-powered web application that detects potato leaf diseases using Deep Learning and Computer Vision.

## Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- OpenCV
- Deep Learning (CNN)

## Features
- Upload an image of a potato leaf
- Real-time disease prediction
- User-friendly Streamlit interface
- End-to-end deep learning pipeline

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the application:
streamlit run app.py

## Project Workflow
- Data Collection
- Image Preprocessing
- Model Training
- Evaluation
- Deployment using Streamlit
